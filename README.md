# 📝ミテモト
スピーチ本番で手元に置くカンペアプリ。話す内容をカード1枚・1分単位で整理し、本番中は暗い画面で大きく表示します。

## 機能
### 編集画面
- スピーチのタイトルを入力
- カードを話す順に並べて作成
- 各カードに「構成メモ」「きっかけ（話し始めの合図）」「キーワード（最大3つ）」を入力
- カードの追加・削除・並び替えが可能
- 入力内容はブラウザのlocalStorageに自動保存
### 本番画面
- 暗い背景で文字を大きく表示
- 前へ／次へボタンでカードを切り替え
- ストップウォッチ機能（本番ボタンを押して3秒後にカウント開始）
- プログレスバーで現在位置を確認
## 使い方
- 編集画面でカードを作成する（構成メモ・きっかけ・キーワードをすべて入力すると本番ボタンが有効になる）
- 「本番」ボタンで本番画面へ
- スピーチ終了後は「終了」で編集画面に戻る
